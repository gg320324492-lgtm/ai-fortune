# AI 算命产品规划

## 一、产品定位

**名称**：AI 智能命理助手（暂定）
**定位**：结合传统命理与 AI 技术的趣味占卜工具
**目标用户**：18-45岁，对星座、命理、八字有兴趣的用户

## 二、功能规划

### 核心功能（免费）
1. AI 塔罗牌 - 每日抽牌解读
2. 星座运势 - 每日/每周/每月运势
3. 名字评分 - 姓名五行、分数分析

### 付费功能（VIP）
4. AI 面相分析 - 上传照片，AI 分析脸型、五官
5. 八字精批 - 出生日期+时间，详细命理分析
6. AI 情感匹配 - 两人配对分析
7. 事业/财运解读 - 职业建议、财运预测

## 三、技术方案

- 前端：Next.js + React + Tailwind CSS
- 后端：Next.js API Routes
- 部署：Vercel / 阿里云
- AI：MiniMax / OpenAI

## 四、变现模式

| 方式 | 价格 |
|------|------|
| 免费次数 | 3次/天 |
| 会员月卡 | 29元/月 |
| 会员年卡 | 199元/年 |
| 单次付费 | 9.9元/次 |

## 五、开发阶段

**第一阶段（1-2周）**：塔罗牌 + 星座运势 + 基础UI
**第二阶段（2-3周）**：名字评分 + AI接入 + 用户登录
**第三阶段（2-3周）**：面相/八字 + 支付接入

## 六、成本

- 域名：50元/年
- 服务器：0元（Vercel免费）
- AI API：200-500元/月
- 初期成本极低
