# AI 命运之书 - 部署指南

## 快速部署到 Vercel

1. 安装 Vercel CLI:
   npm i -g vercel

2. 进入项目目录:
   cd ai-fortune

3. 部署:
   vercel

4. 设置环境变量:
   vercel env add MINIMAX_API_KEY

## 环境变量

- MINIMAX_API_KEY: MiniMax API Key

## 功能

- AI 塔罗牌
- 星座运势
- 姓名分析

## 技术栈

- Next.js 16
- React 19
- Tailwind CSS 4
- MiniMax API
